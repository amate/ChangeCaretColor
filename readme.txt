/* =============================
       ChangeCaretColor
 ============================== */
 
 IMEの状態に応じてキャレットの色を変えます

 TTBaseのプラグインとして動作します
※ 64bit環境で動かす場合、別途64bitのプラグインを読み込めるTTBaseが必要です(peachやTTBaseCppなど)

 Windows10 64bitで動作確認
 

ソフト作成にあたりSetCaretColorを参考にさせてもらいました
また、以下のライブラリを使用しています
